데이터 기반 LDR 연관성 분석

체계적 접근을 통한 반도체 설계 패러다임 혁신

도전 과제: 경험을 넘어 데이터로

Flash Memory 제품 경쟁력의 핵심인 칩 크기 축소(CSR)는 LDR(Layout Design Rule) 최적화에 달려있습니다. 하지만 2만 개가 넘는 방대한 LDR을 기존의 경험 기반 방식으로 분석하는 것은 명확한 한계에 부딪혔습니다.

🤔

높은 복잡도와 인력 의존성

수많은 LDR을 수동으로 분석하는 데에는 막대한 시간과 노력이 필요하며, 결과가 엔지니어의 역량에 크게 의존합니다.

⛓️

"연관 LDR" 문제

하나의 LDR 변경 시 연쇄적으로 영향을 받는 다른 LDR을 놓쳐 개발 일정 지연 및 잠재적 품질 리스크가 발생합니다.

🔄

지식의 단절과 반복

담당자 변경 시 축적된 경험이 단절되어, 매 제품 개발마다 유사한 분석 과정을 비효율적으로 반복하게 됩니다.

우리의 솔루션: PRIME & LEARN 플랫폼

이러한 문제를 해결하기 위해, 정량적이고 자동화된 분석 플랫폼을 개발했습니다. 이 플랫폼은 CSR 영향도 분석(PRIME)과 LDR 연관성 분석(LEARN)의 두 축으로 구성됩니다.

PRIME

CSR 영향도 측정

Calibre DRC/DFM을 활용, 각 LDR의 CSR 잠재력을 정량화하여 핵심 LDR을 선별합니다.

LDR 연관성 예측

머신러닝 모델을 통해 LDR 쌍 간의 연관성을 예측하여 "연관 LDR Chain" 탐색의 기반을 마련합니다.

모델 성능 진화 과정

초기 모델의 불안정한 성능에서 출발하여, 능동 학습 전략의 시행착오와 밸런싱 샘플링을 통해 모델을 안정화하고 성능을 회복시켜 나갔습니다. 이 과정은 데이터의 양뿐만 아니라 질과 균형이 얼마나 중요한지를 보여줍니다.

라벨 추가에 따른 모델 성능 변화

주요 모델 심층 비교

최고 잠재력 vs 최고 안정성 모델

167개 라벨에서 가장 높은 잠재력(AUC)을 보였으나 불안정했고, 269개 라벨에서는 안정성을 확보하며 성능을 회복했습니다.

최종 모델 특징 그룹별 중요도

최종 모델은 수치형 특징에 가장 크게 의존하며, 새로 추가된 Mask 및 CheckType 특징도 의미 있는 기여를 합니다.

핵심 발견 및 교훈

라벨링 전략의 중요성

특정 구간에 편향된 능동 학습은 오히려 모델의 일반화 성능(ROC AUC)을 저하시켰습니다. 데이터의 전체 분포를 반영하는 **균형 잡힌 샘플링**이 모델 안정화와 성능 회복의 열쇠였습니다.

앙상블의 함정

H/V 모델을 분리하고 확률적 OR로 결합하는 앙상블은 의도와 달리 FP(오탐)를 폭증시켜, 'Chain 탐색'이라는 최종 목표에 부적합함을 확인했습니다. 이는 **단일 통합 모델의 성능을 고도화**하는 것이 더 올바른 방향임을 시사합니다.

향후 과제 (The Path Forward)

  • 1

    라벨 데이터 지속 확보

    현재의 긍정적인 회복세를 이어가기 위해, '무작위 샘플링'과 '불확실성 샘플링'을 조합한 균형 잡힌 전략으로 라벨 데이터를 꾸준히 추가하여 모델의 근본적인 성능을 향상시킵니다.

  • 2

    하이퍼파라미터 튜닝

    충분한 라벨 데이터가 확보된 후, 모델의 복잡도를 제어하는 하이퍼파라미터를 최적화하여 현재 특징 세트에서 최대의 성능을 이끌어냅니다.

  • 3

    분류 임계값 최적화

    최종 모델의 예측 확률을 기반으로, 'Chain 탐색' 목표에 가장 적합한 수준의 FP(오탐)를 유지하면서 Recall(재현율)을 극대화하는 최적의 분류 임계값을 설정합니다.
