2. 솔루션: 이중 분석 플랫폼, PRIME & LEARN

기존의 한계를 극복하기 위해, CSR 영향도 분석(PRIME)과 LDR 연관성 분석(LEARN)을 결합한 자동화된 플랫폼을 개발했습니다.

PRIME

Potential Rule Impact Measurement Engine

Calibre DRC/DFM을 활용, 각 LDR의 위반 영역(`Area_rule`)을 정량화하여 **CSR 영향력이 가장 큰 핵심 LDR을 선별**합니다.

LEARN

LDR Exploration using AI-driven Relation Network

PRIME 결과를 바탕으로, ML 모델을 통해 LDR 간의 상호 연관성을 예측하고 **"연관 LDR Chain"을 자동으로 추출**합니다.

3. 모델 성능 진화 과정

모델 성능은 단순히 데이터를 추가하는 것을 넘어, **어떤 특징을 어떻게 추가하고, 라벨링 전략을 어떻게 가져가는지**에 따라 극적으로 변화했습니다. 이 과정은 모델의 근본적인 판별 능력(ROC AUC)과 안정성(표준편차)의 변화를 통해 관찰할 수 있습니다.

라벨 수 증가에 따른 ROC AUC (CV)와 표준편차(에러 바)의 변화 추이

4. 주요 모델 심층 비교 분석

주요 모델 성능 비교

최고 판별력(AUC)을 보인 **167 라벨 모델**과 가장 안정적인 **269 라벨 모델**의 주요 지표 비교. 269 라벨 모델은 안정성은 높지만, 0.5 임계값 기준에서는 Precision이 다소 하락했습니다.

최종 모델 특징 그룹별 중요도

**수치형 특징**이 여전히 예측의 가장 중요한 기반이 되지만, **CheckType**과 **Mask Group** 관련 특징들이 추가되며 모델의 예측 논리가 더욱 정교해졌음을 보여줍니다.

5. 핵심 발견 및 향후 과제

핵심 발견 (Key Findings)

  • **특징의 질 > 라벨의 양:** 단순한 라벨 추가보다, `CheckType`, `Mask Group`과 같은 **문맥 정보를 제공하는 특징 추가**가 모델 성능 개선에 더 큰 영향을 미쳤습니다.
  • **라벨링 전략의 중요성:** 특정 구간에 편중된 능동 학습은 오히려 모델 성능을 저해할 수 있습니다. **전체 데이터 분포를 고려한 균형 잡힌 샘플링**이 모델의 안정성과 일반화 성능에 필수적입니다.
  • **"숨겨진 특징"의 가능성:** 현재 특징 세트로 설명되지 않는 예외 케이스가 존재하며, 이는 논문에서 제안된 바와 같이 더 근본적인 데이터(레이아웃 이미지 등)로부터 **새로운 특징을 발굴해야 할 필요성**을 시사합니다.

향후 과제 (The Path Forward)

  • **새로운 특징 공학:** PRIME에서 계산된 CSR 영향력(`Area_rule`)이나, `err_portion` 지표를 재검토하여 모델에 추가하고 성능을 검증합니다.
  • **모델 최적화:** 충분한 라벨 데이터 확보 후, 하이퍼파라미터 튜닝을 통해 현재 모델의 잠재력을 최대한 끌어냅니다.
  • **운영 지점 설정:** "Chain 탐색"이라는 최종 목표에 가장 적합하도록 **분류 임계값을 조정**하여 Precision과 Recall의 최적 균형점을 찾습니다.
  • **앙상블 모델 재탐색:** 단일 모델 성능이 한계에 도달했을 때, 더 정교한 규칙을 적용한 앙상블 모델을 다시 고려합니다.
